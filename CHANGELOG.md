# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-03

### Added

- Initial release of @authtara/sdk
- AuthClient class for JavaScript SDK
- React components and hooks (AuthProvider, useAuth, useUser, etc.)
- TypeScript support with full type definitions
- Storage abstraction (localStorage, sessionStorage, memory)
- Custom error classes (AuthError, ValidationError, etc.)
- Event system for auth state changes

### Features

- Sign up, sign in, sign out functionality
- User session management
- Pre-built SignIn and SignUp components
- SSR support for Next.js
- Storage abstraction layer
- Comprehensive error handling
